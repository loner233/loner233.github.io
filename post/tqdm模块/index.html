tqdm有三种主要的方法

Iterable-based

把tqdm()加在任何可迭代对象上

1
2
3
4
5
6
7

from tqdm import tqdm
import time

text = ""
for char in tqdm(['a', 'b', 'c', 'd']):
    time.sleep(0.25)
    text = text + char

trange(i)是一个特别的优化了的tqdm(range(i))的实例

1
2

for i in trange(100):
    time.sleep(0.01)

1
2
3
4

pbar = tqdm(["a", "b", "c", "d"])
for char in pbar:
    time.sleep(0.25)
    pbar.set_description("Processing %s" % char)

在循环外面实例化

pbar = tqdm(["a", "b", "c", "d"])

Manual

使用一个with状态来手动控制tqdm的更新

1
2
3
4

with tqdm(total=100) as pbar:
    for i in range(10):
        time.sleep(0.1)
        pbar.update(10)

如果提供了一个可选变量total(或者len()可迭代对象)，预测的过程会被显示出来 当然，with也是可选的，甚至可以指派tqdm()到一个变量上面，但是不要忘记最后del或者close()这个变量

1
2
3
4
5

pbar = tqdm(total=100)
for i in range(10):
    time.sleep(0.1)
    pbar.update(10)
pbar.close()

Module

tqdm可以作为一个单独的脚本在命令行中运行，只需要简单地在管道中插入tqdm或者python -m tqdm

通常tqdm的参数这边也能用

1

find . -name '*.py' -type f -exec cat \{} \; |tqdm --unit loc --unit_scale --total 857366 >> /dev/null
